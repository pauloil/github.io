Windows常用的快捷键

2019-04-28

Windows

常用的快捷键：

三、系统操作快捷键

Alt+Tab 切换应用

Ctrl+Shift 切换输入法

Shift 切换输入法中英文

CapsLock 切换大小写

Win+空格 切换中英文（打游戏最好用的快捷键）

Win+Tab 切换桌面

Win+Shift+C 新建桌面

Win+数字键 切换任务栏的相对应用（1、2、3…）

Win+上键 当前窗口最大化

Win+下键 当前窗口最小化

Win+左键 当前窗口居左（可以按Win+右键还原）

Win+右键 当前窗口居右（可以按Win+左键还原）

Win+W Windows INK工作区

Win+E 打开资源管理器

Win+R 打开运行

Win+T 切换窗口预览

Win+I 打开Windows设置

Win+A 打开通知

Win+D 显示桌面

Win+G 游戏录制

Win+L 锁屏

Win+X 显示系统选项

Win+M 最小化所有窗口

Ctrl+Shift+Esc 任务管理器

四、文件处理快捷键

Ctrl+ C 复制

Ctrl+ X 剪切

Ctrl+ V 粘贴

Ctrl+A 全选

Ctrl+F 查找

Ctrl+Z，撤销操作

Ctrl+Y，还原最近的撤销

Ctrl+N 新建一个新的文件

Ctrl+O 打开“打开文件”对话框

五、编辑文档快捷键

Ctrl+ C 复制

Ctrl+ X 剪切

Ctrl+ V 粘贴

Ctrl+A 全选所有

Ctrl+F 查找

Ctrl+Z，撤销操作

Ctrl+Y，还原最近的撤销

Ctrl+H 查找and替换

Ctrl+B 使字符变为粗体
Ctrl+I 使字符变为斜体
Ctrl+U 为字符添加下划线

Ctrl+P 打印

Shift+F4:重复上次查找命令.
Shift+F5:跳转到文档中的上一次编辑位置

六、浏览器操作快捷键

以下是Chrome的快捷键

1、地址栏：

Ctrl+Enter 在当前输入的文本前加上“www.”在当前输入的文本后加上“.com”，并打开。

Ctrl+Backspace 删除光标前的字词

Shift+Delete 删除自动填充的条目

2、页面和标签页：

Ctrl+W 关闭当前页面

Ctrl+N 打开新窗口

Ctrl+T 打开新标签页

Ctrl+Shift+T 重新打开上次关闭的标签页。

Ctrl+H打开“历史记录”页面。

Alt+Home 在当前窗口打开主页。

Ctrl+Shift+N 在隐身模式下打开新窗口。

Ctrl+1 到 Ctrl+8切换到标签栏中指定位置编号所对应的标签页。

Ctrl+9切换到最后一个标签页。

Ctrl+Tab 或 Ctrl+PgDown切换到下一个标签页。

Ctrl+Shift+Tab 或 Ctrl+PgUp切换到上一个标签页。

3、浏览器快捷键

Alt+F4关闭当前窗口。

Ctrl+J 打开“下载内容”。

Shift+Esc 打开内置任务管理器。

Ctrl+Shift+B 打开和关闭书签栏。

Ctrl+Shift+J 打开开发人员工具。

4、网页快捷键

空格键 向下滚动网页。

Home 转至网页顶部。

End 转至网页底部。

Shift +鼠标滚轮 在网页上横向滚动。

Ctrl+D将当前网页保存为书签。

F11在全屏模式下打开网页。再按一次 F11 可退出全屏模式。

点击鼠标中键（滚轮键）。激活自动滚动模式。

Ctrl+U打开当前网页的源代码。

Ctrl++，Ctrl+-，按住 Ctrl 键并且向上/下滚动鼠标滚轮，放大/缩小网页上的所有内容。

Ctrl+0 将网页上的所有内容都恢复到正常大小。

Ctrl+F打开查找栏。

Ctrl+P 打印当前网页。

Ctrl+S保存当前网页。

F5 或 Ctrl+R重新载入当前网页。

Esc停止载入当前网页。

Ctrl+F5 或 Shift+F5重新载入当前网页，忽略缓存的内容。
