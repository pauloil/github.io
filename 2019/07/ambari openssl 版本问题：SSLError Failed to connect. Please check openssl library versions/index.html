2019-07-13

ambari openssl

ambari openssl 版本问题：SSLError: Failed to connect. Please check openssl library versions.

1.检查os openssl是否是最新版本，若不是，请更新。

2.若不行，修改/etc/python/cert-verification.cfg配置文件

3.重启ambari
